# Texture Assets for Realistic 3D Universities

## Required Textures:
- `brick.jpg` - Brick texture for university buildings
- `glass.jpg` - Glass texture for windows and modern facades  
- `roof.jpg` - Roof texture for building tops

## Recommended Sources:
- **Free**: textures.com, freepbr.com, ambientcg.com
- **Premium**: poliigon.com, quixel.com

## Specifications:
- **Resolution**: 1024x1024 or higher
- **Format**: JPG or PNG
- **Seamless**: Tileable textures preferred
- **PBR**: Physically Based Rendering compatible

## Usage:
These textures are loaded in `RealisticUniversityScene.tsx` to create photorealistic building materials for the 3D campus experience.
